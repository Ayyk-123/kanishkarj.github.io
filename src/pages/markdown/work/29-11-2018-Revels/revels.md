# Revels

Revels is an online digital management system where multiple users can upload, access videos, create channels, playlists and like or comment upon the videos. The project can be used for educational or entertainment purpose.

The project is uploaded on github [here](https://github.com/kanishkarj/DBMS-minor-project).
